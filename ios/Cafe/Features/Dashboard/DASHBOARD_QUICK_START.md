# Dashboard Customization - Quick Start Guide

Get started with customizing your Cafe dashboard in minutes!

## Overview

The Cafe dashboard is now fully customizable. Add, remove, reorder, and configure cards to create your perfect productivity workspace.

## 5-Minute Quick Start

### 1. Access Dashboard Menu (10 seconds)
- Open the Dashboard tab
- Tap the menu icon (⋯) in the top right

### 2. Try a Preset Layout (30 seconds)
- In the menu, tap "Layout Presets"
- Try "Focus" for minimal productivity view
- Try "Overview" for comprehensive dashboard
- Try "Social" for team collaboration

### 3. Add Your First Card (1 minute)
- Tap menu (⋯) → "Customize Cards"
- Tap "Add Card" at the bottom
- Browse available cards by category
- Tap any card to add it
- Tap "Done"

### 4. Reorder Cards (1 minute)
- Tap menu (⋯) → "Edit Layout"
- Notice edit controls appear on cards
- Drag any card to move it
- Tap "Done Editing" when finished

### 5. Configure a Card (2 minutes)
- Enter edit mode (menu → "Edit Layout")
- Tap the gear icon on any card
- Adjust settings:
  - Toggle visibility
  - Change size
  - Set max items
  - Enable auto-hide
- Tap "Done"

## Common Customizations

### Morning Productivity Setup
1. Add these cards (in order):
   - Welcome Header
   - Today's Tasks (large)
   - Calendar
   - Quick Actions
2. Configure Today's Tasks:
   - Size: Large
   - Max Tasks: 10
   - Show Completed: Off
3. Save as "Morning Focus"

### Afternoon Quick Check
1. Add these cards:
   - Welcome Header
   - Overdue Tasks
   - Quick Actions
   - AI Suggestions
2. Configure Overdue Tasks:
   - Auto-hide: On
   - Size: Medium
3. Save as "Afternoon Check-in"

### Evening Wind Down
1. Add these cards:
   - Welcome Header
   - Task Statistics
   - Recent Activity
   - Meal Planning
   - Notes
2. Configure with time-based display:
   - Show only: 17:00 - 23:00
3. Save as "Evening Review"

## Need Help?

### Remove a Card
**Quick way**: Swipe left on card in Customize Cards screen
**Edit mode**: Tap X icon in edit mode

### Reset Everything
Menu (⋯) → "Reset to Default"

### Can't Find a Card Type?
Tap menu → "Customize Cards" → "Add Card"
Browse by category:
- Tasks & Productivity
- Calendar & Events
- Information & Utilities
- Social & Collaboration

### Card Won't Show
Check these settings:
1. Card visibility is ON
2. Auto-hide is OFF (or card has content)
3. Time-based display includes current time
4. Card is in visible cards list
